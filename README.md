# GWM Radio Network TV-Style Starter

This version upgrades the homepage into a fuller ministry broadcast network layout.

## Added in this version
- scrolling ticker bar
- featured broadcasts section
- donation / ministry support block
- announcements area
- stronger TV-style live hero layout
- same live comments, replay archive, schedule, music request, and prayer wall structure

## Run the backend
```bash
cd server
npm install
npm start
```

## Run the frontend
```bash
cd client
npm install
npm run dev
```

## Replace before launch
- stream URL in `client/src/App.jsx`
- donation/support button link
- church email, phone, and address
- archive audio links
- form handling with your real backend or email service
